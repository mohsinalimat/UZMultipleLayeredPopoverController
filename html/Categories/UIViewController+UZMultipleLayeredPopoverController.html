presentMultipleLayeredPopoverWithViewController:contentSize:fromRect:inView:direction:

Present the specified view controller as popover. The popover is always displayed on the unique UZMultipleLayeredPopoverController in the application.

- (void)presentMultipleLayeredPopoverWithViewController:(UIViewController *)viewController contentSize:(CGSize)contentSize fromRect:(CGRect)fromRect inView:(UIView *)inView direction:(UZMultipleLayeredPopoverDirection)direction

Parameters

viewController

The view controller for managing the popover’s content.

contentSize

The new size to apply to the content view.

fromRect

The rectangle in view at which to anchor the popover.

inView

The view containing the anchor rectangle for the popover.

direction

The arrow directions the popover is permitted to use. You can use this value to force the popover to be positioned on a specific side of the rectangle.
